# Product Requirements Document (PRD): WorkFound Europe

## 1. Введение
WorkFound Europe — современная платформа для поиска работы в Европе, ориентированная на рабочие специальности (строители, водители, складские работники, персонал сферы услуг и т.д.). Основная цель — создать прозрачный, удобный и безопасный инструмент для взаимодействия работодателей и соискателей.

### Целевая аудитория
- **Соискатели:** Русскоязычные специалисты, ищущие работу в странах Европы.
- **Работодатели:** Европейские компании и рекрутинговые агентства, заинтересованные в найме персонала.

### Язык платформы
Основной интерфейс: Русский.
(В будущем планируется мульти-язычность, но MVP на русском).

## 2. Ключевые функции и Роли

### 2.1. Соискатель (Worker)
- **Регистрация/Авторизация:** Email, соцсети (Google/LinkedIn).
- **Личный кабинет:**
  - Управление профилем (Фото, ФИО, Контакты).
  - Создание резюме (Опыт работы, навыки, лицензии/права, знание языков).
  - Загрузка документов (Сертификаты, визы).
  - Статус поиска ("Активно ищу", "Рассматриваю предложения").
- **Поиск вакансий:**
  - Фильтры: Страна, Город, Профессия, Зарплата, Тип занятости.
  - Сохранение в избранное.
- **Отклики:** Отправка заявки на вакансию, история откликов и статусы.
- **Уведомления:** Email и внутренние уведомления о новых вакансиях или ответах работодателей.
- **Чат:** Прямая переписка с работодателем.

### 2.2. Работодатель (Employer)
- **Регистрация/Авторизация:** Проверка компании (KYC - Know Your Customer для серьезности).
- **Личный кабинет:**
  - Профиль компании (Логотип, описание, сайт).
  - Управление вакансиями (Создание, редактирование, архивация).
  - Просмотр откликов (Сортировка кандидатов, статусы "Интервью", "Отказ", "Оффер").
  - Поиск резюме (База кандидатов).
- **Продвижение:** (На будущее) Платные опции для поднятия вакансий.
- **Чат:** Прямая переписка с соискателями.

### 2.3. Администратор (Admin)
- Модерация вакансий и профилей.
- Управление пользователями.
- Аналитика платформы.

## 3. Нефункциональные требования
- **Производительность:** Быстрая загрузка страниц (SEO-оптимизация, SSR).
- **Безопасность:** Защита персональных данных (GDPR), защита от спама и ботов.
- **Адаптивность:** Полная поддержка мобильных устройств (Mobile First).
- **UX/UI:** Современный, чистый интерфейс на базе Chakra UI.

## 4. Метрики успеха (KPI)
- Количество активных вакансий.
- Количество успешных наймов (match rate).
- Удержание пользователей (Retention).
